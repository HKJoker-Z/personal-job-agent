# Cover Letter Drafts

Cover Letter generation is an explicit Package action. It uses the saved Job title/company, Match Analysis, and two or three confirmed evidence areas. It does not visit a company website or invent culture, products, news, enthusiasm, or personal connections.

Unknown company context is expressed neutrally. The local grounded seed and optional model rewrite have bounded length and pass strict schema, output-secret scanning, and independent claim validation. Every factual sentence receives an evidence result.

The result is a Draft Material Version. Users may edit it, request changes, explicitly confirm an unresolved claim for that Version, approve it, and finalize it. Nothing is emailed or sent automatically.
